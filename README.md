# Congreso Filosófico: "La Privacidad y Vigilancia en la Era Digital"

**Sitio web oficial del Congreso Filosófico — IE José Celestino Mutis**  
**Desarrollado por:** Miguel Angel Osorio Orduz

---

## Descripción General

Este sitio web fue creado como una plataforma digital para el *Congreso Filosófico* del colegio **Institución Educativa José Celestino Mutis**, con el objetivo de reunir, presentar y exponer diversas ponencias de estudiantes y docentes en torno a temas contemporáneos de filosofía. La página busca ser un espacio accesible, claro y funcional que sirva como archivo digital y punto de encuentro para los participantes y el público en general.

## Tema Central del Congreso

> **"La Privacidad y Vigilancia en la Era Digital"**  
> Un análisis crítico desde la filosofía moderna y contemporánea, abordando las implicaciones éticas y existenciales de la vigilancia digital, la pérdida de privacidad y el papel de la tecnología en la vida humana contemporánea.

---

## Funcionalidades del Sitio

- Página principal con introducción al congreso.
- Secciones de ponencias individuales, organizadas por autores.
- Acceso a documentos y ensayos desarrollados por los ponentes (PDFs descargables).
- Galería de imágenes del evento.
- Diseño responsivo, optimizado para dispositivos móviles y escritorio.
- Estética sobria con enfoque académico.
- Modo oscuro integrado.

---

## Estructura del Repositorio

Este repositorio contiene el sitio estático generado a partir de WordPress usando el plugin **Staatic**, listo para despliegue en cualquier hosting estático (GitHub Pages, Netlify, Vercel, etc.).

```
/
├── index.html                  # Página principal
├── nosotros/                   # Sección sobre el congreso
├── ponencias/                  # Ponencias de los participantes
├── category/                   # Categorías del sitio
├── wp-content/
│   ├── plugins/                # Assets de plugins (CSS/JS)
│   ├── themes/rufous/          # Tema y sus recursos
│   └── uploads/2026/           # Imágenes, PDFs y documentos subidos
├── wp-includes/                # Scripts y estilos base de WordPress
└── sitemap*.xml                # Sitemaps para SEO
```

## Tecnologías Utilizadas

- **WordPress** como CMS de origen
- **Tema Rufous** con Kubio Page Builder
- **Staatic** para la generación del sitio estático
- **Yoast SEO** para optimización de búsqueda
- **WP Dark Mode** para modo oscuro
- HTML, CSS, JavaScript

## Inspiración y Contexto

Este sitio no es solo una vitrina digital, sino una herramienta pedagógica que:

- Democratiza el acceso a la producción filosófica escolar.
- Promueve el uso de herramientas digitales para eventos académicos.
- Plantea preguntas sobre privacidad, vigilancia y el sentido de la existencia digital desde una mirada estudiantil.
